# 🌤️ Weather App

A modern, feature-rich weather application with animated space background and comprehensive weather data.

## 🌐 Live Demo

[View Weather App](https://RavenGames.github.io/weather-app/)

## 🚀 Features

- **Real-time Weather Data**: Current weather conditions using Open-Meteo API
- **Geolocation Support**: Automatic location detection
- **Temperature Conversion**: Toggle between Celsius and Fahrenheit
- **5-Day Forecast**: Extended weather predictions
- **Hourly Forecast**: Detailed hourly weather data
- **Favorites System**: Save and manage favorite cities
- **Animated Space Background**: Beautiful canvas-based starfield animation
- **Responsive Design**: Works perfectly on all devices
- **Dark Theme**: Modern dark UI for better user experience

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **APIs**: Open-Meteo (weather data), Geocoding API
- **Graphics**: HTML5 Canvas for animations
- **Storage**: localStorage for favorites
- **Icons**: Font Awesome weather icons

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Allow location access for automatic weather detection
4. Search for cities or use geolocation

## 🔧 API Information

This app uses the **Open-Meteo API** which provides free weather data without requiring API keys. The API includes:

- Current weather conditions
- 5-day forecast
- Hourly forecast
- Geocoding for city searches
